# PA - Temperature Conversion App

**Course:** SDC340 - Mobile App Development  
**Student:** Patrick Gonzalez  
**Date:** June 27, 2026  

## Description

An Android application that converts temperatures between Fahrenheit and Celsius. The user enters a temperature, selects a conversion direction using radio buttons, and taps the convert button to see the result. Out-of-range values display a Toast error message.

## Features

- Convert °F to °C (valid range: -100°F to 250°F)
- Convert °C to °F (valid range: -75°C to 125°C)
- Toast error messages for out-of-range input
- Thermometer image and clean UI layout

## Tech Stack

- Kotlin
- Android Studio
- ConstraintLayout
- AppCompatActivity

## How to Run

1. Clone the repository
2. Open the project in Android Studio
3. Sync Gradle
4. Run on an emulator or physical device (API 24+)
